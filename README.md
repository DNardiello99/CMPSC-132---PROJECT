~ PROJECT DESCRIPTION ~

This game is for one player. The user is guessing the number that the computer has already picked randomly. The player picks the upper bound for the random number (The difficulty). Once the difficulty is picked, the computer confirms the bounds and then asks the player for a guess. If the guess is too low, it wil print "Too low!". Vice versa, if the guess it too high, it will print "Too high!". Once the player has picked the correct number, the computer will print "Correct!" and display the number attempts it took to reach the answer.
